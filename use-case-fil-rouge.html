S3

Semaine 3 — Creas Publicitaires

3 videos + 3 statiques, strategie 3-2 Dynamic Creative

3 angles × 3 videos + 3 statiques

Angle 1 — Probleme
Hook : "3h du matin. Ton bebe pleure. Tu n'en peux plus."
Video : Scene nuit blanche (lumiere sombre, parent epuise) → decouverte produit → bebe s'endort sous les etoiles
Statique : Split avant/apres — gauche : parent epuise, droite : bebe paisible sous les etoiles
Angle 2 — Benefice
Hook : "Il s'est endormi en 12 minutes. Premiere fois en 6 mois."
Video : Chambre bebe avec projection etoiles, musique douce, bebe calme → parent sourit depuis la porte
Statique : Photo bebe endormi sous projection + texte overlay chiffre ("12 min")
Angle 3 — Preuve sociale
Hook : "14 000 familles l'utilisent deja chaque soir."
Video : Compilation rapide : 3-4 scenes de familles differentes utilisant le produit (videos fournisseur + texte overlay)
Statique : Produit en gros plan + etoiles "4.7/5 — 12 400 avis" + "Satisfait ou rembourse 30j"

Strategie 3-2 — Dynamic Creative

Ad Set 1 — Video
3 videos (Probleme / Benefice / Preuve)
× 2 headings
× 2 textes principaux
= 12 combinaisons
Ad Set 2 — Statique
3 statiques (Probleme / Benefice / Preuve)
× 2 headings
× 2 textes principaux
= 12 combinaisons
Heading 1 : "Offre-lui des nuits etoilees"
Heading 2 : "Bebe s'endort en 12 minutes"
Texte 1 : Axe emotion/douleur (parle du probleme du parent)
Texte 2 : Axe solution/benefice (parle du resultat produit)

S4

Semaine 4 — Lancement META

Lancer la campagne, lire les metriques, decider

Configuration campagne

Objectif
Ventes (Achat)

Pixel deja nourri par les visiteurs S2+S3. Sinon : commencer par "Afficher le contenu" J1-J2.

Budget
10€/jour × 5 jours = 50€

Test de creatifs, pas test produit complet. Objectif : identifier la crea gagnante.

Audience
Large — France + Belgique

Femmes 25-45 ans, aucun interet specifique. L'algo optimise tout seul avec les creatifs.

Structure
1 CBO × 2 Ad Sets

Ad Set 1 = 3 videos DC. Ad Set 2 = 3 statiques DC. META distribue le budget sur le meilleur.

Depense totale : 30€ (J+3 sur 5 jours). 1 vente a 34,90€ = ROAS 1.16x. Pas encore rentable, mais des signaux positifs : bon CTR (1.8%), bon ATC (4.2%), 1 conversion sur budget minimal.

Diagnostic — Decision

Crea gagnante : Video "Probleme" (nuit blanche) — 2.4% CTR, meilleur CPM, seule a generer l'ATC et la vente.

Creatifs coupes : Statique "Preuve sociale" — CTR 0.6%, aucun ATC. Budget gaspille.

Verdict : Produit prometteur. L'angle "douleur parentale" resonne fortement. Pas assez de budget pour conclure, mais les signaux sont positifs.

Plan d'action post-test :
1. Doubler le budget sur la video "Probleme" uniquement (15€/jour)
2. Creer 2 variations de cette video avec des hooks differents
3. Retravailler la page produit : ajouter un timer de livraison + plus de photos lifestyle
4. Negocier le MOQ avec l'agent maintenant que le produit montre des signaux
